Premiers Pas avec Git

Guide d'installation et commandes de base

Installation de Git

  1. Téléchargez Git depuis git-scm.com
  2. Lancez l'installateur et suivez les étapes
  3. Acceptez les options par défaut sauf si vous avez des besoins spécifiques
  4. Vérifiez l'installation en ouvrant Git Bash
brew install git

Ou téléchargez depuis git-scm.com

sudo apt-get install git  # Pour Ubuntu/Debian
sudo yum install git      # Pour Red Hat/Fedora

Configuration initiale

Ces configurations sont stockées dans le fichier ~/.gitconfig

# Configuration de base
git config --global user.name "Votre Nom"
git config --global user.email "votre@email.com"

# Configuration de l'éditeur par défaut
git config --global core.editor "code --wait"  # Pour VS Code

Premières commandes

Initialiser un dépôt

mkdir mon-projet
cd mon-projet
git init

Vérifier l'état du dépôt

git status

Ajouter des fichiers

# Ajouter un fichier spécifique
git add fichier.txt

# Ajouter tous les fichiers
git add .

Créer un commit

git commit -m "Premier commit : initialisation du projet"

Important : Un bon message de commit doit être clair et descriptif. Évitez les messages vagues comme "fix" ou "update".

Exercice pratique

Créez votre premier dépôt

  1. Créez un nouveau dossier pour votre projet
  2. Initialisez un dépôt Git
  3. Créez un fichier README.md
  4. Ajoutez et committez le fichier
  5. Vérifiez l'historique avec git log
